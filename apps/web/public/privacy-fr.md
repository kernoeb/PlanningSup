# Politique de Confidentialité - PlanningSup

*Dernière mise à jour : 18 décembre 2025*

## Données collectées

### Avec un compte (optionnel)
- **Identifiant de compte** : fourni par Discord ou GitHub lors de la connexion
- **Préférences utilisateur** : thème, planning sélectionné, paramètres d'affichage

### Sans compte
- Aucune donnée personnelle n'est collectée
- Les préférences sont stockées localement dans votre navigateur

## Utilisation des données

Les données collectées servent uniquement à :
- Synchroniser vos préférences entre vos appareils
- Maintenir votre session de connexion

## Stockage

- Les données sont stockées sur nos serveurs sécurisés
- Les préférences locales sont stockées dans le localStorage de votre navigateur

## Services tiers

L'authentification utilise OAuth avec :
- **Discord** : [Politique de confidentialité Discord](https://discord.com/privacy)
- **GitHub** : [Politique de confidentialité GitHub](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement)

Nous ne recevons que les informations minimales nécessaires (identifiant utilisateur).

## Analytique

Nous utilisons Plausible Analytics, une solution respectueuse de la vie privée qui ne collecte aucune donnée personnelle et n'utilise pas de cookies.

## Suppression des données

Vous pouvez supprimer votre compte et toutes les données associées à tout moment depuis les paramètres de l'application.

## Contact

Pour toute question concernant cette politique : contact@planningsup.app

## Modifications

Cette politique peut être mise à jour. Les modifications seront publiées sur cette page.
